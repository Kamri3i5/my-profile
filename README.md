# Social Links Profile · Kamron

> Premium personal social links page built from a Frontend Mentor challenge — heavily customized into a real portfolio business card.

🌐 **Live site:** [kamri3i5.github.io/my-profile](https://kamri3i5.github.io/my-profile/)

---

## ✨ Features

- 🎨 **Glassmorphism design** — frosted card with backdrop blur and saturated background
- 🌈 **Animated gradient mesh** — slow-shifting ambient background with grid mask
- ⚡ **Smooth entry animations** — cascading reveal of all elements (cubic-bezier)
- 🔥 **Conic gradient avatar ring** — rotating accent border around profile photo
- 🟢 **Pulsing status indicator** — "available for work" dot with ripple animation
- 💎 **Premium hover states** — lift, glow, color shift, arrow translation
- 📱 **Fully responsive** — mobile-first, tested down to 320px width
- ♿ **Accessible** — semantic HTML, ARIA labels, `prefers-reduced-motion` support, focus-visible outlines
- 🚀 **Zero dependencies** — pure HTML + CSS, ~15KB total

## 🛠 Built with

- Semantic HTML5
- Modern CSS3 (custom properties, backdrop-filter, conic-gradient, mask)
- Google Fonts (Inter + JetBrains Mono)
- Inline SVG icons (no external icon libraries)

## 🚀 Run locally

```bash
git clone https://github.com/Kamri3i5/my-profile.git
cd my-profile
# open index.html in your browser, or use Live Server in VS Code
```

## 🎯 Customization

Want to fork this as your own profile? Edit these in `index.html`:

1. Replace avatar URL (line ~27)
2. Update name, location, tagline, bio
3. Swap social media links in the `<nav class="links">` block
4. Adjust accent colors in `style.css` (`:root` variables at top)
